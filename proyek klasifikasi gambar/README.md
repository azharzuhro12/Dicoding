# Proyek Klasifikasi Sampah 

## Deskripsi
Proyek ini membangun model untuk mengklasifikasikan gambar sampah menjadi 3 kategori:

- **Anorganik(A)** → Botol plastik, kaleng, styrofoam, dll.
- **Bahan Berbahaya dan Beracun(B3)** → Baterai, lampu neon, obat kadaluarsa dll.
- **Organik(O)** → Buah-buahan, sayuran, sisa makanan dll.
